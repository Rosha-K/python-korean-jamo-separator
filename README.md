# python korean jamo separator
입력한 한글의 자모를 분리해서 엑셀에 저장시키는 파이썬 코드입니다.

python korean handler를 수정해서 작성했습니다.

원본 코드는 아래 링크에 있습니다.

https://github.com/neotune/python-korean-handler/

## infomation

한글이 아닌 영문, 특수문자 사용시 작동하지 않습니다.


* `python3.5.X`
